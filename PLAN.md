# Intern Selection Pipeline — Project Plan

## Overview
Automated intern selection pipeline for collecting resumes, building candidate personas, assigning AI-related projects, tracking submissions, and auto-grading them.

## Status: Sprint 1 — Complete ✅

## Tech Stack
- **Backend:** Python + FastAPI + SQLAlchemy + SQLite
- **Frontend:** React + Vite + Tailwind CSS
- **Grading:** LLM (hunter-alpha via OpenRouter) + GitHub API + HTTP checks
- **Hosting:** TBD (local for dev)

## Scoring Weights
| Criteria | Weight | Method |
|---|---|---|
| Deployed Project | 40% | HTTP check → functionality, UI |
| Code Quality | 25% | LLM analyzes repo |
| AI Usage | 20% | LLM checks AI-assisted code |
| Creativity | 15% | LLM evaluates approach |

## Scale: 50-100 candidates

## Candidate Persona Fields
- **Auto-extracted from resume (PDF parser):** Name, email, phone, college, degree, year, skills, projects, work experience
- **Manual (form):** Interests, learning style, availability, motivation, portfolio links, preferred tech stack, past AI tool usage, challenge they solved
- **Resume input:** Upload (PDF/DOCX) OR paste resume link
- **Certifications:** Removed (not required)
- **Generated by LLM:** Skill level, strengths, gaps, assignment fit, risk flags
